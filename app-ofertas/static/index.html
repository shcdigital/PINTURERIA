⚙️ Configuración

Para que la app pueda publicar ofertas necesitás un token de GitHub con acceso al repositorio.

📖 ¿Cómo generar el token?
  1. Andá a github.com/settings/tokens
  2. Click Generate new token (classic)
  3. Dale un nombre (ej: "Gestor Ofertas")
  4. Seleccioná repo (todos los permisos)
  5. Click Generate token
  6. Copiá el token (es la única vez que lo vas a ver)
  7. Pegalo abajo y guardá
